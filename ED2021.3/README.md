# BUILD

> colocar o arquivo de entrada na pasta "src",usar a seguinte linha via terminal na pasta para compilar:
>```
>	g++ -O3 *.cpp* -o main -g
>	```
>  e a seguinte linha para rodar:
>  ```
>	./main <entrada do arquivo CSV> 
>
>  exemplo de linha para execução:
>    ./main t.csv
